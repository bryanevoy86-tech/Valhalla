# Operational Timeline

## Phase 1 — Sandbox Validation
- Isolated environment
- DRY-RUN protections
- Stability and learning verification
- Duration: Ongoing (minimum 6+ hour lockdown before live operation)

## Phase 2 — 44-Week EIA Program
- Real transactions permitted (live environment)
- Conservative throughput
- 10% personal extraction cap
- Majority of revenue reinvested
- Continuous monitoring and adjustment

## Phase 3 — Post-EIA Scaling
- Gradual increase in autonomy
- Controlled expansion
- Continued discipline
- Reinforcement through performance data

---

## Transition Criteria

Each phase requires:
- Successful completion of previous phase
- Risk assessment review
- Control validation
- Stakeholder sign-off
